### 🎓 LearnSync

**An AI-powered adaptive learning platform for Computer Science students.**

LearnSync personalizes learning by tracking each student's mastery of uploaded notes and topics. It generates diagrams, interactive coding exercises with an integrated code editor, and simplifies difficult concepts into easy-to-understand, step-by-step explanations tailored to each learner.

### 🛡️ VeriDoc

**An AI-powered certificate verification system.**

VeriDoc detects forged academic certificates by comparing uploaded documents with authentic institutional templates. It analyzes fonts, typography, spacing, layout, grammar, and document structure to identify inconsistencies and flag potentially fraudulent certificates.

**Tech Stack:** Next.js • TypeScript • Supabase • Gemini API • Squad Payment Webhooks
